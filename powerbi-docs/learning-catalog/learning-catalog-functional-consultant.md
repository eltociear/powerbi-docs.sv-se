# <a name="functional-application-consultant-learning-catalog"></a>Utbildningskatalog för konsulter för funktionella program

Är du implementeringsexpert för en företagsdomän? 

Följande katalog ordnas från kärnkunskaper till specifika domäner och från det grundläggande till det mest avancerade. Om det finns innehåll i flera format meddelar vi dig, så att du kan välja det utbildningsformat som bäst motsvarar dina behov.

För funktionella konsulter visade vår analys av jobbaktiviteter att alla måste förstå både kärninnehållsuppsättningen och ett domänområde.
